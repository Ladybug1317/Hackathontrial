What is the problem?

Electric consumption, specifically wastage of electricity.

Electricity is a kind of resource that many think can be used constantly without repercussions. There are repercussions. Electricity usage in high amounts can cause a lot of carbon emissions, even though it doesn't seem like it. Using electricity mindlessly causes us to unintentionally pollute the environment, we need to stop using it unnecessarily.
